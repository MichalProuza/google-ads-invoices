# Google Ads Invoice Automator

Automatically downloads Google Ads PDF invoices for the previous month and emails them via Gmail API. Runs monthly via GitHub Actions.

---

## How it works

1. On the 5th of each month, GitHub Actions triggers `fetch_invoices.py`
2. The script connects to Google Ads API and fetches invoices for each configured account
3. PDFs are downloaded and sent as email attachments via Gmail API

---

## Setup

### 1. Google Ads API credentials

1. Enable the **Google Ads API** in [Google Cloud Console](https://console.cloud.google.com/)
2. Create an **OAuth 2.0 Client ID** (Desktop app)
3. Apply for a **Developer Token** in your Google Ads account → Tools → API Center
4. Generate a refresh token using the [OAuth Playground](https://developers.google.com/oauthplayground/) or a local script

You'll need these values for GitHub Secrets:
- `GOOGLE_ADS_DEVELOPER_TOKEN`
- `GOOGLE_ADS_CLIENT_ID`
- `GOOGLE_ADS_CLIENT_SECRET`
- `GOOGLE_ADS_REFRESH_TOKEN`

### 2. Gmail API credentials

1. Enable the **Gmail API** in Google Cloud Console
2. Create an **OAuth 2.0 Client ID** (Desktop app) and download the JSON
3. Run the token generator locally:

```bash
pip install -r requirements.txt google-auth-oauthlib
GMAIL_CREDENTIALS_FILE=client_secret.json python generate_token.py
```

4. Copy the printed JSON into the `GMAIL_TOKEN_JSON` GitHub Secret
5. Copy the contents of `client_secret.json` into `GMAIL_CREDENTIALS_JSON` GitHub Secret

### 3. GitHub Secrets

Go to your repo → **Settings → Secrets and variables → Actions** and add:

| Secret | Description |
|---|---|
| `GOOGLE_ADS_CUSTOMER_IDS` | Comma-separated account IDs, e.g. `1234567890,9876543210` |
| `GOOGLE_ADS_DEVELOPER_TOKEN` | Google Ads developer token |
| `GOOGLE_ADS_CLIENT_ID` | OAuth2 client ID |
| `GOOGLE_ADS_CLIENT_SECRET` | OAuth2 client secret |
| `GOOGLE_ADS_REFRESH_TOKEN` | OAuth2 refresh token for Ads API |
| `GMAIL_CREDENTIALS_JSON` | Contents of the Gmail OAuth2 client secret JSON |
| `GMAIL_TOKEN_JSON` | Token JSON generated by `generate_token.py` |
| `SENDER_EMAIL` | Gmail address to send from |
| `RECIPIENT_EMAIL` | Email address to receive invoices |

## Manual run

Trigger the workflow manually from **GitHub Actions → Monthly Google Ads Invoices → Run workflow**.

## File structure

```
├── fetch_invoices.py          # Main script – downloads invoices and triggers email
├── send_email.py              # Gmail API sender module
├── generate_token.py          # One-time local helper to generate Gmail token
├── requirements.txt
└── .github/
    └── workflows/
        └── monthly-invoices.yml   # GitHub Actions cron workflow
```
